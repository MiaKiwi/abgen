# ABGEN

> Abbreviations Generator, for generating abbreviations!

ABGEN is a DUMB (Doesn't Use Mediocre Bartificial-intelligence, yes I came up with that myself) abbreviation generator. 

Don't feel like coming up with acronyms like DONUT (DON't Use Technology) or initialisms like HUH (Historically Unlawful Hospital)??? Well you're in luck! Welcome to ABGEN, the simple abbreviation generator fueled by nothing but a poor understanding of english grammar, countless nights writing documentation, and a desire to vary how I creates abbreviations for my stuff.

### Abbreviations

Although I call them abbreviations, the correct term would actually be "abbreviated terms" ([ISO style guide](https://www.iso.org/ISO-house-style.html#:~:text=Abbreviated%20terms%2C%20abbreviations%20and%20contractions)). However, I am an adult and have chosen to ignore that.

### Work in progress

This project is VERY work-in-progress. You can of course contribute by extending the ``BaseAbbreviator`` class and registering your generator in the generator register.
